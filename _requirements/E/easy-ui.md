<div class="quality-requirement" markdown="1">

#### Context

Users need to be able to register their smartphone as a device for two-factor authentication.

#### Trigger

User wants to register smartphone for two-factor authentication.

#### Acceptance Criteria

- Configuration setting for adding smartphone as 2FA device is found within 10 minutes
- 95% of users can access the configuration settings in 10 minutes or less
